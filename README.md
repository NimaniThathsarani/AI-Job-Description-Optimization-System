# AI Job Description Optimization System

## Overview

This project analyzes job descriptions to identify factors that improve applicant engagement and application rates.

The system uses Natural Language Processing (NLP), readability analysis, sentiment analysis, keyword extraction, and machine learning to evaluate and optimize job postings.

## Features

- Job description effectiveness analysis
- Readability scoring
- Sentiment analysis
- Keyword extraction using TF-IDF
- Application prediction model
- Interactive Tableau dashboard

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- TextBlob
- TextStat
- Tableau
- Git & GitHub
